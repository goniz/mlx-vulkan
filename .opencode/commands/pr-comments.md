---
description: Fetch and address PR review comments for mlx submodule
---

Fetch the non-resolved PR review comments for the mlx/ submodule by running `./dev.sh pr-comments --submodule mlx`. Then review each unresolved comment and address them in the code. For each comment, either:
1. Fix the issue mentioned in the comment
2. Reply to the comment explaining what you changed or why it doesn't need to be addressed

Make sure to resolve or address all comments.
